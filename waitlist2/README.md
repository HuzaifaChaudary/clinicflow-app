# CLINICFLOW

A dark, premium, architectural marketing website for Clinicflow—a healthcare SaaS platform for clinic operations.

## Design Philosophy

**Dark · Premium · Futuristic · Architectural**

This website embodies:
- Deep, architectural color palette (#0A0D12, #0F141B)
- Glass morphism with heavy blur (32px+) and subtle borders
- Editorial typography using Inter
- Soft mint accents (#5EEAD4) used as radial glows, not fills
- Slow, intentional animations
- Editorial imagery with dark overlays (60-80%)

## Pages

1. **Home** - Powerful hero, brand positioning, problems to solutions
2. **Product** - Core capabilities explained conceptually
3. **How It Works** - Step-based implementation process
4. **Solutions** - By clinic type (Mental Health, Physio, Dental, Outpatient)
5. **Voice Automation** - Communication features
6. **Pricing** - Transparent, trust-focused
7. **About** - Mission, philosophy, values
8. **Contact** - Minimal form with reassuring copy

## Technical Stack

- **React 18** with TypeScript
- **React Router** for navigation
- **Motion** (Framer Motion) for animations
- **Tailwind CSS 4.0** for styling
- **Vite** for build tooling

## Key Features

- Non-sticky navigation that appears on scroll
- Large glass panels as structural elements
- Radial gradient glows for accent lighting
- Editorial image treatment with overlays and blur
- Slow, heavy animations (1.2s+ durations)
- Responsive design with architectural proportions

## Development

```bash
npm install
npm run dev
```

## Design Principles

**Avoid:**
- Bright colors or high-energy gradients
- Small cards or repetitive layouts
- Stock photography of people
- Bouncy or playful animations
- Sticky headers
- Obvious SaaS templates

**Embrace:**
- Space and negative space
- Architectural composition
- Editorial restraint
- Depth through light and shadow
- Intentional motion
- Calm confidence
