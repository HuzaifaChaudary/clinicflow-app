### `/dashboard` - Clinic Management Dashboard
The internal dashboard application for clinic staff, doctors, and administrators to manage appointments, patients, schedules, and clinic operations.

**Key Features:**
- Role-based dashboards (Admin, Doctor, Owner)
- Appointment scheduling and management
- Patient intake automation
- Voice AI integration
- Real-time activity tracking
- Form builder for custom intake forms
- Settings and configuration management

**Tech Stack:**
- React + TypeScript
- Vite
- Tailwind CSS
- Radix UI components

### `/web` - Public Website
The public-facing website for marketing and patient-facing features.

**Key Features:**
- Marketing pages (Home, Product, Solutions, Pricing)
- Voice automation demonstrations
- Clinic-specific solutions (Dental, Mental Health, Physiotherapy, Outpatient)
- Contact and trial signup
- Operational flow visualizations

**Tech Stack:**
- React + TypeScript
- Vite
- Tailwind CSS
- Custom UI components
